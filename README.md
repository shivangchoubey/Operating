# 💻 CPU Scheduling Algorithm Simulator 🌈  

![Demo Banner](https://imgur.com/xqpo8BH.jpg)  

## ✨ Features  
✅ **6 Scheduling Algorithms** (FCFS, SJF, Round Robin, etc.)  
✅ **Interactive Gantt Charts**  
✅ **Real-Time Metrics**  

## 🚀 Quick Start  
1. Clone this repo  
2. Run `python app.py`  
3. Open `http://localhost:5000`
